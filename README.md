# GameAIexps
游戏AI之前做的一些实验的整理和汇总。
## 环境需要：
* TensorFlow 1.x
* OpenCV
* pygame
* matplotlib
* gym (只能在Linux系统上)
搭环境的教程可以看：[CUDA和cuDNN安装教程](https://blog.csdn.net/Gerzson/article/details/94652816)和[Ubuntu上搭游戏环境教程](https://blog.csdn.net/Gerzson/article/details/91478123)
*为了整理，更改了一些文件的相对路径，注意把代码里面的相对路径改过来。
